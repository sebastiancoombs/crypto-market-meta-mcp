Crypto Market META MCP

Pay-per-call x402 META-aggregator for cross-feed crypto pricing. Composes 5 already-shipped MT services (coinpaprika + coincap + coinlore + cryptocompare + kraken) in parallel into one consensus price (median + mean + spread). Single billable hit per price-discovery query instead of N. Pay USDC on Base via x402. No signup.

x402 USDC on Base META-aggregator consensus no signup

Method + PathPriceDescription
